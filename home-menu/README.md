# 家庭菜品管理 · 小鸟系统（手机友好）

一个在本地浏览器里运行的「家庭菜品 / 材料 / 计划」小系统，重点适配手机浏览。

## 功能概览

- **菜品管理 `/dishes`**
  - 字段：菜品名称、菜品预览图链接、菜品分类（带彩色 icon）、主厨、材料、步骤（支持新增多步）。
  - 多选菜品 + 选择日期，生成对应日期的 **「今日菜谱」**。
  - 如果该日期已经有菜谱，再次生成时会给出覆盖提示。

- **今日菜谱 `/menus/today?date=YYYY-MM-DD`**
  - 展示当天所有菜品卡片（材料 + 步骤），布局适合手机**截图保存成图片**。

- **材料管理 `/ingredients`**
  - 材料单独表单：名称、数量、单位、有效期。
  - 「密封 x3」按钮：将**剩余有效期延长 3 倍**，并标记为已密封。

- **计划管理 `/plan`**
  - 日历视图（周 / 月 / 年切换），用小彩色 icon 标记每天的菜谱。
  - 点击某一天，跳转到对应的「今日菜谱」。

## 启动方式

在终端中执行：

```bash
cd /Users/finvol/Documents/home-menu
python3 server.py
```

然后在浏览器（电脑或手机）打开：

- 菜品管理：`http://127.0.0.1:8010/dishes`
- 今日菜谱：`http://127.0.0.1:8010/menus/today`
- 材料管理：`http://127.0.0.1:8010/ingredients`
- 计划管理：`http://127.0.0.1:8010/plan`

> 提示：手机上也可以直接在浏览器里打开这些地址（需要同一局域网，并把 `HOME_MENU_HOST` 改为 `0.0.0.0`）。

```bash
HOME_MENU_HOST=0.0.0.0 python3 server.py
```
